Pure Chess Library Documentation

Welcome to PureChess Docs!

This is the official documentation page of the Pure Chess Engine. PureChessLib is a .NET library that provides the basic logic of the game of chess.

Installation

To use our chess engine, follow the installation instructions available here. This version does not come with a GUI (Graphical User Interface), you can use external tools for that, or use our own: OpenChessTools

Setting Up

Before you start using Pure Chess Library, make sure you have a basic understanding of C# and .NET development. Here's how you can set up the library in your project:

  1. Download the latest release of Pure Chess Library from the GitHub repository.
  2. Add the PureChessLib assembly to your project's references.
  3. Import the PureChess namespace in your code files: using PureChess;
  4. You're now ready to start using the library in your chess-related applications.

Examples of Usage

Let's take a look at some simple examples of how to use Pure Chess Library to create a basic chess game:

// Create a new chess board
Board board = new Board();
board.GenerateBoard(null);

// Load a custom position
string customPosition = "rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR";
board.LoadPosition(customPosition);

// Make a move
Move move = new Move();
move.InitialSquare = board.Squares[8]; // Example initial square
move.TargetSquare = board.Squares[16]; // Example target square
bool isValidMove = engine.MakeMove(move);

// Display the updated position
board.DrawCurrentPosition();

API and Methods

The Pure Chess Library API provides various methods to interact with the chess engine. Here are some of the most commonly used methods:

  • GenerateBoard(string position): Generates the chess board based on the provided position.
  • LoadPosition(string positionToLoad): Loads a specified position onto the board.
  • MakeMove(Move move): Makes a move on the board.
  • DrawCurrentPosition(): Draws the current position of the chess board.

Frequently Asked Questions

Here are some frequently asked questions about Pure Chess Library:

  • Q: Can I use this library for commercial projects?
    A: Yes, Pure Chess Library is open-source and available under the MIT License, which allows you to use it in commercial projects.
  • Q: How can I contribute to the development of this library?
    A: You can contribute by submitting bug reports, feature requests, or pull requests on the GitHub repository.
  • Q: Is there a graphical interface included with this library?
    A: No, Pure Chess Library provides the basic logic of chess. You can use external tools or build your own GUI.
