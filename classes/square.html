Square Class

The Square class represents an individual square on the chess board. It contains information about the square's position, its associated piece, and methods to retrieve coordinates.

Properties

  • Index - Gets the index of the square.
  • X - Gets the X-coordinate of the square.
  • Y - Gets the Y-coordinate of the square.
  • Piece - Gets or sets the piece associated with the square.

Methods

  • GetCoord() - Returns the coordinate of the square in algebraic notation.

Examples

// Create a new square instance
Square square = new Square();
square.Index = 0;
square.X = 0;
square.Y = 0;

// Set a piece on the square
Piece piece = new Piece();
piece.Type = "Pawn";
piece.Color = 0;
piece.Value = 1;
square.Piece = piece;

// Get the coordinate of the square
string coord = square.GetCoord(); // Returns "a1"

For more details, refer to the source code and documentation of the Square class.
