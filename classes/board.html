Board Class

The Board class is responsible for managing the chess board, including generating the board, loading positions, making moves, and displaying the board's current state.

Constructor

Board() - Creates a new instance of the Board class.

Methods

  • GenerateBoard(string position) - Generates the chess board and loads the specified position.
  • LoadPosition(string positionToLoad) - Loads the specified position on the board.
  • LookForKings() - Locates the positions of the kings on the board.

Properties

  • DefaultPosition - Gets or sets the default position of the chess board.
  • Columns - Gets the number of columns on the board.
  • Rows - Gets the number of rows on the board.

Examples

// Create a new chess board instance
Board board = new Board();
board.GenerateBoard(null);

// Load a custom position
string customPosition = "rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR";
board.LoadPosition(customPosition);

// Display the current board position
board.DrawCurrentPosition();
