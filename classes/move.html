Pure Chess Library Documentation

Move Class

The Move class represents a chess move, containing information about the initial and target squares of the move.

Properties

  • InitialSquare - Gets or sets the initial square of the move.
  • TargetSquare - Gets or sets the target square of the move.

Methods

  • AddToMoveList() - Adds the move to the list of moves and updates the PGN notation.

Examples

// Create a new move instance
Move move = new Move();
move.InitialSquare = new Square();
move.TargetSquare = new Square();

// Add the move to the move list
move.AddToMoveList();

For more details, refer to the source code and documentation of the Move class.
