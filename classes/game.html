Pure Chess Library Documentation

Game Class

The Game class in the Pure Chess Library manages the overall game state, player turns, and game settings.

Properties

playerTurn: number

Represents the current player's turn. It can have two values: 0 for white player's turn and 1 for black player's turn.

state: GameState

An enumeration that represents the current state of the game. It can have values like GameState.NotStarted, GameState.Playing, GameState.Checkmate, and GameState.Stalemate.

moves: Move[]

An array that holds all the moves made during the game. Each move is an instance of the Move struct.

gamePGN: string

A string that represents the game in PGN (Portable Game Notation) format. PGN is a standard notation used to record chess games. The format includes information about the moves, result, and other game-related data.

Methods

StartNewGame(): void

Starts a new chess game by initializing the board, setting initial player turns, and resetting the moves history.

StopGame(): void

Stops the current game, changing its state to GameState.NotStarted. This method is typically used to end a game prematurely or reset the game state for a new game.

RestartGame(): void

Restarts the current game by resetting the board and player turns. This is useful for restarting a game after it has concluded.
