Piece Class

The Piece class represents a chess piece with properties such as type, color, and value. It provides methods to update and retrieve piece information.

Properties

  • Type - Gets or sets the type of the piece (e.g., "Pawn", "Rook").
  • Color - Gets or sets the color of the piece (0 for white, 1 for black).
  • Value - Gets or sets the value of the piece.

Methods

  • GetPieceName() - Returns the name of the piece including its color.
  • GetPieceType(inputType) - Checks if the piece is of a specific type.
  • UpdatePieceInfo() - Updates piece information based on its type and color.

Examples

// Create a new piece instance
Piece piece = new Piece();
piece.Type = "Rook";
piece.Color = 0;
piece.Value = 5;

// Get the piece name
string pieceName = piece.GetPieceName(); // Returns "White Rook"

// Check if the piece is a Pawn
bool isPawn = piece.GetPieceType("Pawn"); // Returns false
